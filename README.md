# Employee Management System

## Instructions

### Instructions for the Frontend

1. Clone the porject
2. cd ./frontend-erp
3. run command `npm i`
4. run command `npm start`

### Instructions for the Backend

1. cd ./backtend-erp
2. run command `npm i`
3. run command `npm start`

## Platform

Frontend = React JS <br />
Backend = Node JS
